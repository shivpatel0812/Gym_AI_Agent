# GymAI React Native Frontend

React Native TypeScript frontend for the GymAI fitness tracking app.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Configure environment variables in `app.json` or use Expo environment variables

3. Run the app:
```bash
npm start
```

Then press `i` for iOS simulator or `a` for Android emulator.

## Project Structure

- `App.tsx` - Main app with navigation
- `src/components/` - All React Native components
  - `workouts/` - Modular workout components
  - `wellness/` - Modular wellness components
  - Other feature components

All components use React Native StyleSheet for styling.
